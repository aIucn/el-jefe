Key Terms From Chapter 2

External style sheet
A separate text document linked to from the webpage to style it.
ID selector
A CSS selector beginning with # used once per webpage.
Inline style
Formatting applied directly within an HTML tag.
List item
An item within an ordered or unordered list.
Nested
HTML tags created within other HTML tags.
Ordered list
HTML tags used to create numbered or lettered lists.
Relative links
Links that do not include the full file path.
Unordered lists
HTML tags used to create bulleted lists.
Version control
Software for creating timestamped copies of projects.
WYSIWYG
An editor showing "What You See Is What You Get."
